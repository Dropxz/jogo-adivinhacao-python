# Jogo de Adivinhação em Python

Projeto simples desenvolvido durante meus estudos de Python.

## Sobre o projeto
O objetivo é adivinhar uma palavra secreta digitando uma letra por vez.

## Tecnologias
- Python

## Aprendizados utilizados
- Estruturas de repetição (while, for)
- Manipulação de strings
- Lógica de programação
